*The Self-Healing Foundation for Hyperpolymath Hardware.*

== Overview

HardwareResilience.jl provides the "Guardian" logic for long-running kernels. It detects hardware faults and handles transparent fallbacks.

== Features

* **KernelGuardian**: Monitor and track the status of critical kernels.
* **Self-Healing**: Automated error handling and fallback logic.
